# VoltCar Desk

Одностраничное приложение для учёта заказ-нарядов и записей клиентов (frontend CRA + backend Express/SQLite).

## Структура репозитория
- `client/` — фронтенд (Create React App, TypeScript, Tailwind, React Router, JWT).
- `server/` — бэкенд (Express, JWT, SQLite/Prisma, PDF‑генерация).
- `server/data/` — локальные данные (БД, загруженные PDF) — не версионируются.
